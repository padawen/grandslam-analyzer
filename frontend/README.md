# Grand Slam Analyzer - Frontend

Vue 3 + Vite frontend for visualizing tennis betting strategies.

## Setup
```bash
npm install
```

## Environment Variables
Copy `.env.example` to `.env` and configure:
- `VITE_API_URL`: Backend API URL (default: `http://localhost:8000`)
- `VITE_API_KEY`: API key matching the backend

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

Frontend will be available at `http://localhost:5173`
